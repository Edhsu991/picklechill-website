# Venue Photos

Put venue guide photos in this folder.

Recommended filenames:

- `venue-entrance-wide.jpg` - 東寧路上的網球場入口外觀
- `tennis-court-gate.jpg` - 台南一中網球場門牌入口
- `court-side-walkway.jpg` - 進入後的場邊走道
- `left-turn-passage.jpg` - 左轉後往底走的通道
- `pickleball-court-view.jpg` - 走到底後的匹克球場

Use paths like `assets/venue/pickleball-court-view.jpg` in `src/data/siteContent.js`.
